A showcase of web development projects demonstrating front-end design and functionality.

## Projects

A food delivery or meal management application. [View Project](dabbewale.html)

### 🏥 HealBuddy

A health and wellness companion application. [View Project](healbuddy.html)

### 💧 SipSentry

A beverage or hydration tracking application. [View Project](sipsentry.html)

## Technologies Used

- **HTML5** - Semantic markup and structure
- **CSS3** - Styling and responsive design
- **JavaScript** - Interactivity and dynamic functionality

## Features

- Responsive web design
- Interactive user interfaces
- Clean and modern layouts
- Cross-browser compatibility

## How to Use

1. Clone this repository

   ```bash
   git clone <repository-url>
   ```

2. Open `index.html` in your web browser to view the portfolio homepage

3. Navigate through the projects from the main page
