Overview

HealBuddy ML Edition is a multipage Streamlit app featuring three machine-learning classification models for local, interpretable predictions. It pairs model output with SHAP and LIME explanations.

Key Features

  • Decision tree, random forest and SVM classification models
  • SHAP and LIME integration for explainable predictions
  • Multipage Streamlit workflow for model interaction
  • Local prediction flow without relying on remote APIs

Tech Stack

  • Programming Language: Python
  • App Framework: Streamlit
  • Modeling: Decision Tree, Random Forest, SVM
  • Explainability: SHAP, LIME
