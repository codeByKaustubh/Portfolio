Overview

HealBuddy is a browser-based symptom checker that uses deterministic rule logic to map symptom patterns to common health conditions. It is designed for general awareness, not clinical diagnosis.

Key Features

  • Guided symptom input with rule-based evaluation
  • Condition suggestions based on symptom pattern matching
  • Responsive client-side experience for mobile and desktop

Tech Stack

  • Frontend: HTML, CSS, Vanilla JavaScript
  • Logic: Client-side deterministic rules
