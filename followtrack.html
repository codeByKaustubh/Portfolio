FollowTrack

Client-side Instagram follower analysis tool

Home Contact

Overview

FollowTrack parses user-exported Instagram JSON data locally to identify non-followers and asymmetric connections. It avoids server calls, third-party API dependencies, credential collection and scraping.

Key Features

  • Upload Instagram follower and following JSON exports
  • Detect non-followers and one-sided follow relationships
  • Display follower and following statistics
  • Secure flow with no login, scraping or remote processing
  • Fast comparison using set-based algorithms

Tech Stack

  • Programming Language: Python
  • App Framework: Streamlit
  • Logic: JSON parsing and set-difference comparison
