Overview

Built a privacy-first Instagram analytics tool using Python and Streamlit to identify non-followers and asymmetric connections. Implemented JSON data parsing and set-based algorithms while ensuring secure, credential-free operation.

Key Features

  • Upload Instagram data files (JSON)
  • Detect non-followers
  • Identify one-sided follow relationships
  • Display follower & following statistics
  • Secure — no login, no scraping
  • Fast processing using Python sets

Tech Stack

  • Programming Language: Python
  • Frontend: Streamlit
  • Logic: Compare followers and following using set difference
