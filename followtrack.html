Overview

HealBuddy is a rule-based symptom checker that helps users identify common possible conditions based on reported symptoms. It's intended as an informational tool and not a replacement for professional medical advice.

Key Features

  • Guided symptom input with follow-up questions
  • Rule-based inference engine
  • Responsive UI for mobile and desktop

Tech Stack

  • Frontend: HTML, CSS, Vanilla JavaScript
  • Logic: Client-side rule evaluation
