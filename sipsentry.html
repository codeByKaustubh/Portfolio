Overview

SipSentry is a lightweight Progressive Web App that delivers customizable hydration reminders through browser notifications. It is installable, offline-capable and does not require a native app.

Key Features

  • Customizable hydration reminders
  • Browser notification support
  • Installable PWA experience
  • Offline-capable, lightweight implementation

Tech Stack

  • Frontend: HTML, CSS, JavaScript
  • PWA: Service Worker, browser notifications
  • Storage: Local browser storage
