Overview

SipSentry is a lightweight Progressive Web App that helps users maintain hydration by sending periodic reminders and tracking intake.

Key Features

  • Reminder notifications (PWA)
  • Simple intake tracking
  • Offline-capable design

Tech Stack

  • Frontend: HTML/CSS/JS, Service Worker
  • Storage: LocalStorage / IndexedDB
