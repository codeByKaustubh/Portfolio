Overview

DabbeWale is a platform that connects local home-cooked meal providers with customers in the area. It focuses on simple listings, ordering, and supporting small cooks in the community.

Key Features

  • Provider listings with menus and ratings
  • Simple ordering flow
  • Mobile-first design

Tech Stack

  • Frontend: React / Bootstrap
  • Backend: Node.js (APIs)
